std::string : C++ STL 에서 제공하는 문자열을 다루기 위한 클래스

기존 C 스타일의 문자열 문제점
- 메모리 관리의 어려움(메모리 오버플로우 : 메모리보다 큰 데이터를 넣을 시)
- 복잡한 문자열 연산(ex 문자열 결합 시 새로운 메모리 할당 후 복사하는 등)
- 안정성(널 종료 문자 의존)
- 성능개선(메모리 할당 관련)

주요 특징
1. 동적 메모리 관리
 : std::string은 내부적으로 메모리를 동적으로 할당하고 관리함 -> 문자열 길이를 고려 X
2. 널 종료 문자 필요 없음
 : 내부적으로 크기를 추적
3. 편리한 조작 가능
 : 문자열 결합 연산자, 추출, 찾기 등 기본 메서드나 연사자 제공
4. 안정성
 : 메모리 관리 등을 자동으로 처리하여 버퍼 오버플로우 등의 오류를 예방
